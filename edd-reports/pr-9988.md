This is a case where domain knowledge about KCAPI's internal patterns actively misled the investigation away from a straightforward buffer overflow fix.

Additionally, wolfSSL error code conventions (BAD_FUNC_ARG vs ECC_BAD_ARG_E) are underdocumented, causing both Claudes to potentially use wrong codes.
